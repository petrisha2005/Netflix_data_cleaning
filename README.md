# Netflix Data Cleaning Project

This project demonstrates cleaning and preparation of the Netflix Movies & TV Shows dataset.

## Files Included
- `Raw_Data.xlsx` → Original dataset (unchanged)
- `cleaned_dataset.xlsx` → Cleaned dataset with Data Quality Notes
- `cleaned_dataset.csv` → CSV version of cleaned dataset

## Cleaning Steps
- Missing director and cast names filled
- Missing country values handled
- date_added standardized (DD-MMM-YYYY)
- rating missing values filled
- Duration_Type derived from duration column
- Listed_in and description preserved

## Notes
- Cleaning decisions documented in `Working_Clean` sheet
- Data ready for analysis and reporting
- Version-controlled via Git and hosted on GitHub
